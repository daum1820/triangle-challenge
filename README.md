### Triangle Challenge 
###### by Tradeshift

#### Environment
* Java 8
* Maven
* Redis

#### IDE
* IntelliJ

#### Server Side
* Spring Boot
* Spring Boot MVC
* Spring Boot Security
* Spring Boot Zuul
* Spring Boot Redis

#### Client Side
* Bootstrap 3.3.7
* AngularJS 1.6.4
* Bower Package Manager

#### Tests
* JUnit
* Cucumber

#### Commands
###### Clean
```shell
$ mvn clean
```
###### Clean
```Test
$ mvn test
```
###### Build
```shell
$ mvn package
```
###### Run
```shell
$ mvn spring-boot:run
```

The application will be running on this address: [http:localhost:8080/tradeshift](http:localhost:8080/tradeshift)
